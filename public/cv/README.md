# CV Upload Instructions

## How to Add Your CV

1. **Save your CV** with a clear filename, for example:
   - `Victor-Musyoka-CV.pdf`
   - `resume.pdf`
   - `cv.pdf`

2. **Place the file** in this directory (`public/cv/`)

3. **The CV will be accessible** at the URL: `/cv/your-filename.pdf`

## Recommended Format

- **File type**: PDF (most professional and universally compatible)
- **File size**: Keep under 2MB for fast downloads
- **Filename**: Use hyphens instead of spaces (e.g., `Victor-Musyoka-CV.pdf`)

## Example

If you upload a file named `Victor-Musyoka-CV.pdf`, it will be downloadable at:
`https://your-domain.com/cv/Victor-Musyoka-CV.pdf`

## Next Steps

After uploading your CV here, the download links throughout your portfolio will automatically work.
